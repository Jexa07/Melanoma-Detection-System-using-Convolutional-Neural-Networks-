## 📌 Project Description

This project presents a complete deep learning pipeline for detecting and segmenting **melanoma cells** in histopathological whole-slide images using **YOLOv11m-seg** — a state-of-the-art variant of the YOLO architecture tailored for segmentation.

> **Input Format:** 256×256 pixel tiles extracted from `.SVS` (Aperio) slides  
> **Platform:** Google Colab (GPU-enabled with Drive integration)

---

## ✨ Key Highlights

- 🎯 **Accurate melanoma cell detection** using advanced segmentation  
- ⚡ **Fast & scalable inference** for real-time medical diagnostics  
- 🧪 **Annotation-based training** with extensive data augmentation  
- ☁️ **Cloud-native implementation** for accessible training + testing
